### 4　入力した数を表示したあと降順にソートして表示
入力
```
9
1
34
2
6
7
5
32
12
```
出力結果は
```
9,2,1,34,2,6,7,5,32,12
34,32,12,9,7,6,5,2,2,1
```
- 配列のソートについて
配列を`x[N]`、サイズを`N`とします<br>
最初のfor文`for(int a=0;a<N-1;a++)`で比べる要素を動かします。<br>
次のfor文`for(int b=a+1;b<N;b++)`で比べられる要素を動かします。<br>
if文の
```
if(x[a]<x[b]){
  int tmp=x[b];
  x[b]=x[a];
  x[a]=tmp;
```
で比較と入れ替えを行っています。要素を入れ替えるには同じ型の変数が必要なのでここでは`tmp`を使っています。<br>
昇順ソートにするならif文の条件式を`x[a]<x[b]`から`x[b]>x[a]`に変えると実装可能です。


次に`pC=pB`で`pC`が`&b`が指している変数をさせるようにしています。<br>
最後に`*pC*=2`で`&b`が指している値を2倍して代入してます。<br>
実は`c`にも`&c`にも何も手が加えられていないので何も変わらないんです。
